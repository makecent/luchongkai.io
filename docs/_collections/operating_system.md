## Basic ##

## Files ##

## Terminal ##
`Preferences -> Shortcuts`: view all terminal shortcuts

`ctrl + alt + T`: create a new terminal

`shift` + `CtrlC`/`CtrlV`: copy and paste in terminal

`ctrl + U`: delete whole line

`xkill`: use cursor to select a window to kill.
